# 肯 (Ken) 确反指南

> [!info] 什么是确反？
> 当对手使用了有较大收招硬直（负帧）的招式被我们防御后，我们可以用必定命中的连段进行反击，称为"确反"。

本指南按照对手招式的**不利帧数**分类。请根据防御后的距离和帧数选择合适的反击。

---

## 帧数速查

| 不利帧 | 推荐起手 | 核心连段 | 伤害参考 | 备注 |
|:---:|:---|:---|:---:|:---|
| **-4** | `2MK` | `2MK > KK > 623P（跑升）` | 1950 | 轻攻击极限，2MK 距离比 5LP 靠谱 |
| **-5** | `5LK` 🔥 | `5LK > 623HP（重升龙）` | 1480 | 肯的招牌确反，距离"长的一批" |
| **-6** | `2MP` | `2MP > 2MK > KK > 623P（跑升）` | 2700 | 板中标准；板边用 `5MP > 236HK（重迅雷） > 颠球` |
| **-7** | `2MK` | `2MK > KK > 623P（跑升）` | 1960 | 下段起手，注意距离 |
| **-8+（大确反）** | `2MP` 或 `5HP` | `2MP > 214LK（轻旋风） > 623MP（中升龙）` | 3130 | **不要手软！** |
| **远距离 -6~-8** | `SA2（236236K）` | 直出 SA2 | 3360 | 详见 [[07 - 资源运用与进阶/SA2与OD升龙压制\|SA2 章节]] |

## -4 帧确反
对手招式被防后 -4f，这是大多数轻攻击的极限。

| 推荐起手 | 连段示例 | 伤害 （约） | 备注 |
| :--- | :--- | :--- | :--- |
| **2MK （下中脚）** | `2MK > KK > 623P（跑升）` | 1950 | 虽然 5LP （站轻拳） 发生最快，但距离太短容易打空。**现阶段推荐用 2MK 衔接确反**。 |
| **5LP （站轻拳）** | `5LP > ...` | - | 理论上 +8 有利，但距离是硬伤。 |

## -5 帧确反
对手招式被防后 -5f。肯的 5LK 距离非常优秀。

> [!tip] 核心技：5LK
> 肯的 `5LK` （站轻脚） 发生 4f，距离“长的一批”，是 -5 确反的神技。

### 不消耗斗气 （0气）
| 连段 | 伤害 | 备注 |
| :--- | :--- | :--- |
| `5LK > 623HP（重升龙）` | 1480 | **最稳确反**，带 +25 压制。 |
| `5LK > 大旋风` | 1400 | 推板效果好，+36 压制。 |

### 带资源 （必杀/SA）
| 连段 | 伤害 | 备注 |
| :--- | :--- | :--- |
| `5LK > 236KK（OD迅雷） > 重派生 > SA2` | 3100 | 消耗 2 斗气 + 2 气。 |
| `5LK > 623HP（重升龙） > SA3` | 3400 | 倒地 +25。 |
| `5LK > SA3` | 3560 | 简单直接。 |

## -6 帧确反
对手招式被防后 -6f。可以使用 `2MP` （下中拳） 起手。

### 常用选择
| 连段 | 伤害 | 场景 |
| :--- | :--- | :--- |
| `2MP > 2MK > KK > 623P（跑升）` | 2700 | **板中标准**。 |
| `2MP > 2MK > 大旋风` | 2140 | 不消耗资源的简单选择。 |
| `5MP > 236HK（重迅雷） > 颠球` | 2980 | **板边推荐**。 |

### 带必杀 (SA)
| 连段 | 伤害 | 备注 |
| :--- | :--- | :--- |
| `2MP > 跑踢 > SA1` | 3240 | - |
| `TC > 龙尾 > SA1` | 3387 | 取消一次。 |
| `TC > 龙尾 > SA2` | 3907 | - |

## -7 帧确反
对手招式被防后 -7f。`2MK` （下中脚） 是主力，因为它也是下段。

### 推荐连段
| 连段 | 伤害 | 备注 |
| :--- | :--- | :--- |
| `2MK > KK > 623P（跑升）` | 1960 | 基础确反。 |
| `2MK > 大旋风` | 1400 | - |
| `2MK > 236KK（OD迅雷） > 重派生` | 1940 | 消耗 2 斗气，后续可安全跳 (+42)。 |

## -8 帧及以上 （大确反）
对手招式被防后 -8f 或更多（如升龙挥空）。这是输出的黄金机会。

> [!danger] 不要手软！
> 遇到大确反机会（如防住对方升龙），请务必使用高伤害连段（PC连）。

### 常用起手
- **板中**: `2MP` 起手 或 `5HP` （站重拳） 起手。
- **板边**: `TC` (5MP>5HP) 起手 或 `5MP>2HP`。

### 0资源高伤
| 连段 | 伤害 | 备注 |
| :--- | :--- | :--- |
| `2MP > KK > 623P（跑升）` | 3020 | 板中基础。 |
| `2MP > 214LK（轻旋风） > 623MP（中升龙）` | 3130 | 稍微复杂一点。 |
| `TC > 236MK > 6HK（中迅雷重脚派生） > 623HP（重升龙）` | 3200 | 板边基础。 |
| `17f 双颠球` | 3380 | 板边进阶。 |
### 远距离特殊确反
有时候防住对方的招式距离很远，普通拳脚打不到。
- **SA2（`236236K`）**: 发生 6f，距离远，可以直接确反很多 -6/-7/-8 的远距离招式（伤害 3360）。详见 [[07 - 资源运用与进阶/SA2与OD升龙压制|SA2 用法章节]]。
- **5MK（站中脚）**: 距离远，可接 TC 或 2MK 跑升。
