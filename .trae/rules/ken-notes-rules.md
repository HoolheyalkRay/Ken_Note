# Street Fighter 6 Notation Rules

This project follows strict formatting rules for Street Fighter 6 terminology to ensure consistency across all notes.

## 1. Instruction Terms (UPPERCASE)
All inputs, command normals, and special move inputs involving buttons must be written in **UPPERCASE**.

- **Format**: `[Direction][Button]`
- **Examples**:
  - `2mk` → **2MK**
  - `5hp` → **5HP**
  - `jhk` → **JHK**
  - `236p` → **236P**
  - `lp` → **LP**

## 2. Acronyms (UPPERCASE)
Common mechanics acronyms must be **UPPERCASE**.

- **Examples**:
  - `od` → **OD** (Overdrive)
  - `sa` → **SA** (Super Art)
  - `tc` → **TC** (Target Combo)
  - `di` → **DI** (Drive Impact)
  - `dr` → **DR** (Drive Rush)
  - `pc` → **PC** (Punish Counter)

## 3. Proper Nouns & Mechanics (Title Case)
Specific mechanics and named concepts should be **Title Case** (Capitalize the first letter).

- **Examples**:
  - `shimmy` → **Shimmy**
  - `oki` → **Oki** (Okizeme)
  - `safe jump` → **Safe Jump**
  - `frame trap` → **Frame Trap**
  - `whiff` → **Whiff**
  - `burnout` → **Burnout**
  - `punish guide` → **Punish Guide**
  - `whiff punish` → **Whiff Punish**
  - `hadoken` → **Hadoken**
  - `drive rush` → **Drive Rush**
  - `light punch` → **Light Punch**
  - `medium punch` → **Medium Punch**
  - `heavy punch` → **Heavy Punch**
  - `light kick` → **Light Kick**
  - `medium kick` → **Medium Kick**
  - `heavy kick` → **Heavy Kick**
  - `drive gauge` → **Drive Gauge**
  - `super art` → **Super Art**
  - `overdrive` → **Overdrive**
  - `target combo` → **Target Combo**
  - `drive impact` → **Drive Impact**
  - `reversal` → **Reversal**
  - `tech throw` → **Tech Throw**
  - `link` → **Link**
  - `punish counter` → **Punish Counter**
  - `counter` → **Counter**
  - `okizeme` → **Okizeme**

## 4. Headlines & Titles (Title Case)
English text within headers (starting with `#`) must follow **Title Case** rules (Capitalize the first letter of each word), except for strict Instruction Terms and Acronyms which remain Uppercase.

- **Examples**:
  - `# STARTERS & CONFIRMS` → `# Starters & Confirms`
  - `# punish guide` → `# Punish Guide`
